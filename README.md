# PDA Group 10 Project - Exploratory Data Analysis

## Overview
This project contains our comprehensive Exploratory Data Analysis (EDA) for the PDA course. The analysis focuses on intentional homicide data with extensive statistical analysis and visualizations.

## Project Structure
```
├── Group 10/                  # Analysis outputs and data files
│   ├── EDA Group 5.ipynb      # Main analysis notebook
│   ├── cleaned_intentional_homicide.csv
│   └── various output files (q*.csv)
├── notebooks/                  # Jupyter notebooks
├── outputs/                   # Final reports and visualizations
│   ├── final_report_1300.md   # Comprehensive final report
│   ├── presentation.pptx      # Project presentation
│   └── various plots (*.png)
├── analytics.txt              # Analysis notes
├── rubric.txt                # Project requirements
└── various utility scripts (*.py)
```

## Key Features
- Comprehensive data cleaning and preprocessing
- Statistical analysis of homicide trends
- Regional and temporal analysis
- Gender-based analysis
- Age group analysis
- Outlier detection and analysis
- Multiple visualization types

## Main Analysis File
The primary analysis is contained in `Group 10/EDA Group 5.ipynb` which includes:
- Data loading and cleaning
- Exploratory data analysis
- Statistical summaries
- Visualization generation
- Regional comparisons
- Trend analysis

## Outputs
- **Final Report**: `outputs/final_report_1300.md` - Complete analysis report
- **Visualizations**: Various PNG files in the `outputs/` directory
- **Processed Data**: Clean datasets in CSV format
- **Presentation**: PowerPoint presentation summarizing findings

## Technologies Used
- Python
- Pandas for data manipulation
- Matplotlib/Seaborn for visualization
- Jupyter Notebooks for analysis
- Statistical analysis libraries

Group 10 - PDA Course

1. Open the Jupyter notebooks to explore the analysis
